# Affiliate AutoPoster Documentation

## Purpose
This folder is the no-code, implementation-ready documentation pack for Affiliate AutoPoster.

It is based on the source plan:
- `/home/yasva/Downloads/college/affiliate-autoposter-plan.md`

## Scope and Depth
- V1: Deep documentation (build-ready)
- V2 to V4: Concise delta documentation
- Separate tracks for frontend, backend, and cloud/devops
- Includes full product/business suite

## Structure
- `00-overview/`: vision, roadmap, glossary
- `01-phase-plan/`: phase-wise plan documents
- `02-frontend/`: frontend architecture and delivery docs
- `03-backend/`: backend architecture, APIs, schema, security, scheduler
- `04-cloud-devops/`: deployment, CI/CD, operations, scaling path
- `05-product-business/`: PRD, personas, pricing, KPI, compliance
- `06-quality/`: test strategy, acceptance criteria, UAT checklists
- `99-appendix/`: source mapping and open decisions

## How to Use
1. Read `00-overview/product-vision.md` and `00-overview/roadmap-summary-v1-v4.md`.
2. Execute by phase from `01-phase-plan/`.
3. Build each engineering stream using `02-frontend/`, `03-backend/`, and `04-cloud-devops/`.
4. Validate against `06-quality/acceptance-criteria.md` and `06-quality/uat-checklists.md`.

## Out of Scope
- Source code
- Infrastructure provisioning scripts
- Automated test scripts
