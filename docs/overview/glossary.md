# Glossary

- Affiliate Tag: User identifier appended to product links for commission tracking.
- Amazon PA API: Product Advertising API used to fetch product/deal metadata.
- Post Attempt: One publish operation to a platform; stored with success/failure status.
- Quartz Job: Scheduled backend job that triggers publishing at configured times.
- Platform Post ID: Identifier returned by Telegram/Pinterest/etc after successful post.
- TTL Cache: Time-bound cache (Redis) used to reduce repeated external API calls.
- Multi-Tenant: Architecture where multiple customers are isolated within the same system.
- UAT: User Acceptance Testing based on business scenarios.
