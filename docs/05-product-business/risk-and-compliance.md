# Risk and Compliance

## Key Risks
- External API policy or quota limits
- Credential leakage risk
- Incorrect affiliate link generation
- Platform posting failure spikes

## Compliance Considerations
- Follow affiliate platform terms (Amazon, later Pinterest/Meta/X)
- Respect platform anti-spam and posting policies
- Maintain privacy and secure handling of user integration secrets

## Risk Controls
- Encrypt sensitive credentials
- Implement retry with guardrails
- Maintain audit logs and incident runbooks
- Clear user messaging for configuration errors
