# User Personas and Journey

## Persona 1: Solo Affiliate Starter
- Goal: earn side income with minimal daily effort
- Needs: simple setup, trusted automation, clear status visibility

## Persona 2: Content Channel Operator
- Goal: maintain consistent channel activity
- Needs: scheduled posting and category control

## Persona 3: Growth-Oriented Marketer (V3+)
- Goal: maximize click-through across platforms
- Needs: multi-platform posting and analytics

## Core Journey (V1)
1. Sign up and authenticate.
2. Add Amazon affiliate and Telegram settings.
3. Choose categories/filters and schedule.
4. Receive automatic posts.
5. Track posting outcomes on dashboard.
