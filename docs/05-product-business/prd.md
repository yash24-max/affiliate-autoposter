# Product Requirements Document (PRD)

## Product
Affiliate AutoPoster

## Goal
Enable users to automate affiliate posting and increase consistent commission opportunities.

## Primary Problem
Users cannot maintain daily manual affiliate promotion workflow.

## V1 Requirements
- User authentication and onboarding
- Affiliate source and publishing config
- Automated posting schedule
- Basic performance dashboard

## Non-Functional Requirements
- Secure credentials handling
- Reliable scheduled posting
- Clear error feedback to users

## Release Objective
Launch MVP with enough reliability to onboard first paying users and collect usage feedback.
