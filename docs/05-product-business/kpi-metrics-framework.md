# KPI and Metrics Framework

## Activation Metrics
- Sign-up to configuration completion rate
- Configuration completion to first successful post rate

## Engagement Metrics
- Active schedules per day
- Posts per active user
- Dashboard return frequency

## Reliability Metrics
- Post success ratio
- Scheduler job failure ratio
- External API failure contribution

## Business Metrics
- Free to paid conversion
- Monthly recurring revenue
- Churn rate by plan

## V1 Target Focus
Prioritize activation and posting reliability before advanced monetization optimization.
