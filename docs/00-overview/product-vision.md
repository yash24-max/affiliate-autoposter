# Product Vision

## Problem
Affiliate creators fail to earn consistently due to manual product hunting, design effort, and posting inconsistency.

## Vision
Provide a SaaS platform where users connect affiliate and publishing channels once, then auto-post high-quality affiliate content on schedule.

## Core Value
- For users: passive affiliate earnings via automation
- For platform owner: recurring subscription revenue

## Target Users
- Individual affiliate marketers
- Content creators with Telegram/Pinterest audience
- Side-hustle users with limited daily time

## Value Propositions
- Simple setup: connect Amazon + Telegram and start
- Consistent posting without manual work
- Measurable outcomes: post volume, clicks, estimated earnings

## V1 Success Definition
- User can sign up, configure Amazon + Telegram, set schedule, and receive automated posts.
- Dashboard shows posts and status correctly.
