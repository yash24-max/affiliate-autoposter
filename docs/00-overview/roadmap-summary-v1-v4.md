# Roadmap Summary (V1 to V4)

## V1: Core MVP (Weeks 1-8)
- Auth (JWT + Google OAuth)
- Amazon PA API fetcher
- Telegram publisher
- Quartz per-user scheduling
- Basic dashboard
- Docker + initial deployment

## V2: Visual Layer + Pinterest
- HTML to PNG post templates
- Cloudinary image storage
- Pinterest publisher
- Template selection
- Additional product source support

## V3: Multi-Platform + AI
- Instagram, Twitter/X, WhatsApp, Facebook publishing
- AI product selection
- Click tracking and short links
- Event-driven pipeline with Kafka
- Analytics store with ClickHouse

## V4: Full SaaS Scale
- Payments and subscriptions
- Multi-account and agency mode
- White-label option
- Mobile dashboard app
- AWS production stack + Kubernetes
