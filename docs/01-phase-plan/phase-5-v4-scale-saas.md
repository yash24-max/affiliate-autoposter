# Phase 5: V4 SaaS Scale

## Objective
Transform the product into a scalable, commercial multi-tenant SaaS.

## Scope (Delta from V3)
- Subscription billing and plan enforcement
- Multi-account and agency workflows
- White-label capability
- Production observability and SRE runbooks
- AWS-first deployment topology with orchestration

## Non-Goals
- Major redesign of V1 user journey
- New affiliate networks outside planned roadmap

## Exit Criteria
- Plan restrictions enforce feature access correctly
- Operational dashboards and alerts cover critical paths
- Recovery procedure for DB and queue failures is documented
