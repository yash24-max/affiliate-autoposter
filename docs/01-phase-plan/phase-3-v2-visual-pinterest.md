# Phase 3: V2 Visual and Pinterest Expansion

## Objective
Improve post quality and long-tail traffic by adding generated image templates and Pinterest publishing.

## Scope (Delta from V1)
- HTML/CSS template system for product creatives
- Render pipeline (headless browser)
- Cloud image storage and retrieval
- Pinterest publisher integration
- Template selection per user

## Non-Goals
- Full analytics redesign
- Multi-platform expansion beyond Pinterest

## Exit Criteria
- Posts can include generated image assets
- Pinterest post flow works for configured users
- Template selection is persisted and respected
