# Affiliate Autoposter

This repository contains the source code and documentation for the Affiliate Autoposter project.

## Directory Structure

- `docs/`: Comprehensive project documentation, plans, and specifications.
- `backend/`: API services and backend logic.
- `frontend/`: User interface and frontend application.
- `infra/`: Infrastructure-as-code and deployment configurations.
- `shared/`: Shared libraries and types across frontend and backend.
