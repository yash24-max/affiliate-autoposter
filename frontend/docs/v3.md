# Frontend V3 — Delta (Multi-Platform + AI)

This document covers only what changes or is added in V3. All V1 and V2 features remain. See [v1.md](v1.md) for the full baseline.

---

## New Screens

### Click Analytics
- New dashboard tab: click events per post, by platform, by country, by device
- Charts: line chart (clicks over time), bar chart (clicks by platform)
- Source: `GET /api/analytics/clicks`

### AI Config
- New settings section: enable/disable AI-assisted product picking
- Toggle AI picker on/off per user
- Optional: set preference context (season, trending topics)
- Endpoint: `PUT /api/user/ai-preference`

### Multi-Platform Toggles
- Platform selection in schedule settings: Telegram, Pinterest, Instagram, Twitter/X, WhatsApp
- Each platform can be individually toggled active/inactive
- Plan gate: PRO unlocks Pinterest; AGENCY unlocks all
- Short URL preview per post shown in recent posts table

---

## New API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/analytics/clicks` | Click events with filters (platform, date range) |
| GET | `/api/analytics/clicks/summary` | Aggregated click stats |
| GET | `/api/user/ai-preference` | Get AI picker setting |
| PUT | `/api/user/ai-preference` | Enable/disable AI picker |
| GET | `/api/platforms` | List available platforms and their activation status |
| PUT | `/api/platforms/{platform}/toggle` | Enable or disable a specific platform |

## UX Notes

- AI picker toggle shows a description tooltip explaining what it does
