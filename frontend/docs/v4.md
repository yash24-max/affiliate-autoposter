# Frontend V4 — Delta (Full SaaS Scale)

This document covers only what changes or is added in V4. All prior version features remain. See [v1.md](v1.md) for the full baseline.

---

## New Screens

### Subscription and Billing
- Plan selection page: FREE / PRO / AGENCY cards with feature comparison
- Payment flow integrating Razorpay (India) or Stripe (global)
- Current plan status + renewal date shown in account settings
- Upgrade/downgrade prompts triggered by plan-gated feature access attempts

### Agency View
- Dashboard switcher: toggle between managed client accounts
- Client list: add/remove clients, view per-client posting status
- Team member management: invite team members, assign roles (viewer/editor/admin)
- Bulk actions: activate/deactivate schedules across multiple accounts

### White-Label Settings
- Custom domain configuration
- Logo and color theme upload
- Custom branding preview (AGENCY plan only)

---

## Mobile App Note

A React Native mobile dashboard app is planned for V4 as a separate repository. It consumes the same API Gateway endpoints as the web frontend. Authentication uses the same JWT flow. The mobile app is not part of this repository — refer to the mobile repo once created.

---

## New API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/api/billing/plan` | Get current plan and subscription status |
| POST | `/api/billing/upgrade` | Initiate plan upgrade (returns payment session) |
| GET | `/api/agency/clients` | List managed client accounts |
| POST | `/api/agency/clients` | Add a client account |
| DELETE | `/api/agency/clients/{id}` | Remove a client account |
| GET | `/api/agency/team` | List team members |
| POST | `/api/agency/team/invite` | Invite a team member |
| GET | `/api/whitelabel/config` | Get white-label branding config |
| PUT | `/api/whitelabel/config` | Update white-label branding |

## UX Notes

- Subscription page accessible from account settings and from plan-gate prompts
- Agency view only visible when user plan is AGENCY
- White-label settings only editable by account owner, not team members
- Plan enforcement messages are friendly and include a direct upgrade CTA
